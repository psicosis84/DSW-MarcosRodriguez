Propósito del sitio

El propósito de este sitio es dinamizar el aprendizaje del tema "Etiquetas Básicas de HTML, atributos y valores", mediante herramientes que permitan mejorar la comprensión de cada uno de los elementos básicos en el diseño de una página web.

pantalla inicial

El HTML es un markup language, significa que es un lenguaje que se puede leer sin que sea necesario compilarlo antes. Estos códigos marcan el contenido de la página para ser mostrado por un navegador web.

Cuando se escribe en HTML, se están escribiendo etiquetas HTML en donde cada una está conformada por partes específicas, incluyendo:

  • El carácter “menor que” <
  • Una palabra o carácter que determina qué etiqueta se está escribiendo
  • Cualquier número de atributos HTML que se quiera usar, escritos de la forma nombre =” valor”
  • El carácter “mayor que” >

    • El Hipertexto HTML es un lenguaje informático utilizado por la mayoría de páginas web y plataformas online, por lo tanto, no es considerado como un lenguaje de programación, sin embargo, los usuarios de éste lenguaje pueden usar elementos, etiquetas y atributos para crear párrafos y enlaces que den forma a la página que se está construyendo.

    Con el paso de los años y el avance significativo de la tecnología, el HTML ha evolucionado para pasar de 18 etiquetas iniciales a las 142 etiquetas disponibles hoy día, aunque, algunos de los ordenadores nuevos ya no admiten algunas de ellas. La actualización más importante fue la introducción en 2014 del HTML5, la principal diferencia entre ambas es que la nueva admite nuevos tipos de controles de formularios, incluyendo diversas etiquetas semánticas que describían mejor el contenido, como text text < article >, < headers > y < footer >.

    Los elementos HTML suelen dividirse en tres partes. Esta combinación de las tres crea un elemento HTML.

  • Etiqueta de apertura: Se utiliza para indicar dónde empieza un elemento. Está envuelta en corchete de apertura y cierre. Por ejemplo, puedes usar la etiqueta de inicio < p > para crear un párrafo.
  • Contenido: El contenido es el resultado que ve la audiencia.
  • Etiqueta de cierre: Es lo mismo que la etiqueta de apertura, pero con una barra inclinada delante del nombre del elemento. Es decir, < /p > para finalizar un párrafo.
